### 🛒 MultShop - Laboratório de E-commerce

Este repositório é o meu espaço dedicado ao estudo e desenvolvimento de uma API para sites de vendas. O foco aqui é colocar a mão na massa, testar novas funcionalidades do ecossistema Java/Spring e evoluir constantemente a arquitetura do projeto conforme vou aprendendo novas técnicas.

### 🚀 O que estou desenvolvendo e treinando:

| Funcionalidade | Status | O que estou praticando |
| :--- | :---: | :--- |
| **Gestão de Produtos** | ✅ | CRUD completo e lógica de categorias. |
| **Persistência de Dados** | ✅ | Integração com banco de dados via Spring Data JPA. |
| **Carrinho de Compras** | 🏗️ | Lógica de negócio e manipulação de estados. |
| **Autenticação (Security)** | 📅 | Planejando implementar Spring Security com JWT. |

### 🚀 Tecnologias e Ferramentas em Uso:

- **Linguagem:** Java 11+.
- **Framework:** Spring Boot.
- **Banco de Dados:** MySQL / PostgreSQL.
- **Gerenciador:** Maven.
- **Integração:** Spring Data JPA.

### 📥 Como Rodar o Projeto (Para Testes):
Para ver o progresso atual do meu laboratório, você precisará do JDK 11+ instalado:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/May-SG/MultShop.git
   ```
   
2. **Execute a aplicação:**
  ```bash
  mvn spring-boot:run
  ```
